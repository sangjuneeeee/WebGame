# 미완성 프로젝트
Bulletproof React 아키텍쳐를 따라 구조를 설정하였고, zod를 사용하여 클라이언트에서도 회원가입 및 로그인의 유효성 검사를 진행하고 StyleX를 이용하여 런타임 내에 스타일이 정해지는 것이 아니라 컴파일 타임에 정해지도록 스타일링 하였으며 회원가입 및 로그인 폼까지 완성하였다.
